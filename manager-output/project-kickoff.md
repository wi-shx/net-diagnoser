### 关键约束

**技术约束**:
- 开发语言: Python 3.10+
- AI模型: GLM-4.7 / GLM-5.0
- SSH库: Paramiko
- 时间约束: MVP开发 2周

**功能约束**:
- 日志类型: dmesg + 定制化
- 不包含: nginx/haproxy、证书、DNS
- SSH执行: 端到端闭环（非仅建议）

**部署约束**:
- 操作系统: Ubuntu 20.04+, CentOS 7+
- 界面: CLI优先，暂不实现Web UI
- 部署: 单机部署
